# stm32-hal-touch-cst816t-driver
这是一个为STM32微控制器开发的、针对CST816T电容触摸芯片的专用驱动程序。以硬件iic为通信，hal库为基础，可移植keil5
它的核心目标是提供一套清晰、稳定、易于集成的软件接口。让开发者能在STM32平台上高效地实现触摸屏功能

分层清晰：采用“应用层-驱动层-硬件层”架构，I2C通信和延时等底层操作被封装，便于移植到不同硬件平台。

功能完整：不仅实现了基础的坐标读取，还支持硬件复位、芯片ID与固件版本识别、手势识别（单击、长按、滑动等）、休眠与唤醒等功能。

鲁棒性强：代码中包含完善的参数检查、错误处理和超时机制，能有效防止因硬件异常导致的程序崩溃。

配置灵活：允许开发者通过初始化配置，设置中断模式、自动复位时间、长按判定阈值等参数

硬件依赖：需要STM32的HAL库支持，通过I2C总线（如I2C1）与CST816T通信，并使用至少一个GPIO引脚（如PB8）进行芯片复位，一个GPIO口的中断引脚可选

使用方式：直接将c文件和h文件下载到工程，代码有注释根据硬件接口改动对应的接口
